📈 持股清單編輯

股票代號 股票名稱 均價（元） 市場 操作

📖 操作說明

  1. 首次使用請填入 GitHub Token 並按儲存，Token 僅存在瀏覽器本地，不會上傳。
  2. 頁面載入時會自動從 GitHub 讀取目前持股清單。
  3. 新增、修改、刪除完成後按「儲存到 GitHub」，Actions 下次執行時就會套用新清單。

⚠️ 填寫注意事項

  • 股票代號是識別唯一股票的依據，請務必正確填寫，填錯會導致法人資料抓不到（報告顯示「無資料」但不會報錯）。
  • 代號格式：4〜6碼，只能含數字和英文字母（例如：2330、00878、00981B）。
  • 均價請填寫你的實際持股成本，單位為元。
  • 上市選 twse，上櫃選 tpex。
